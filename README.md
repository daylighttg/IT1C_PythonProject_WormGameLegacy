# IT1C_PythonProject_WormGameLegacy
# 🐍 Classic Worm Game - Python Legacy
**Course:** IT1C **Project Type:** Final Project

## 📌 PROJECT OVERVIEW
This project is a digital recreation of the classic **"Worm" or "Snake Game"**, developed entirely using Python programming language. The game follows all traditional mechanics: the player controls a continuously moving worm, consumes food to grow longer, increases score, and avoids collision with walls or its own body.

This project demonstrates fundamental programming concepts including variables, loops, conditional statements, functions, and data structures using built-in Python modules.



## 🛠️ TECHNOLOGIES & LIBRARIES USED
- **Python 3.x** - Core programming language
- **Turtle Module** - Used for graphics, shapes, movement, and screen rendering
- **Time Module** - Controls game speed and delay
- **Random Module** - Generates random positions for food spawning
- **Winsound Module** - Adds audio feedback for game events




## 🎮 GAME FEATURES
✅ **Classic Gameplay** – Authentic movement and mechanics
✅ **Dual Controls** – Play using `W A S D` keys or `Arrow Keys`
✅ **Score System** – Tracks current score and highest score achieved
✅ **Difficulty Progression** – Game speeds up automatically as score increases
✅ **Audio Feedback** – Sounds when eating food or crashing
✅ **Game Over System** – Detects wall collision and self-collision
✅ **Restart & Exit Option** – Press `[R]` to play again, `[E]` to exit
✅ **Background Support** – Custom background image support (falls back to solid color if missing)

# GAME RULES & CONTROLS
Objective: Eat the red food items to gain points and grow longer.
Controls:
W / ↑ : Move Up
S / ↓ : Move Down
A / ← : Move Left
D / → : Move Right
R : Restart game after Game Over
E : Exit and close the game

Game Over Conditions:
Colliding with the boundary/wall
Colliding with your own body
